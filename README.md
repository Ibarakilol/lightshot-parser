# Lightshot Parser

Lightshot random image parser.

## Usage
1. Clone repository
```
$ git clone https://github.com/Ibarakilol/lightshot-parser.git
$ cd lightshot-parser
```
2. Install dependencies
```
$ yarn
```
3. Change in main function call number of images to download
```
$ main(10)
```
4. Run script
```
$ npm run start
```
